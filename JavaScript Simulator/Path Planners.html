Path Planner Simulation

NOTE: Currently A-Star / Greedy Best First Search use the Euclidean Heuristic, however multiple options are there in the code. Just one map has been created for this simulator as of now, however, you can create multiple maps of your own using the Python Based Simulator in the repository. You can alter the default start and goal points by modifying the URL:
../Path-Planning-Simulator/JavaScript Simulator/Path Planners.html?q_init=[0.0,0.5]?q_goal=[4.3,4.4]
